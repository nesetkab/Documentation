Package com.pedropathing.localization.localizers

Class TwoWheelLocalizer

  • All Implemented Interfaces:
    
public class TwoWheelLocalizer
extends Localizer

    This is the TwoWheelLocalizer class. This class extends the Localizer superclass and is a localizer that uses the two wheel odometry with IMU set up. The diagram below, which is modified from Road Runner, shows a typical set up.
